Project: Task Manager App

A full-stack web app to manage tasks.

Frontend:
- React + Vite
- Tailwind CSS

Backend:
- Node.js + Express
- REST API
- JWT authentication

Features:
- Register / Login
- Add / Edit / Delete tasks
- Task status (todo, in-progress, done)
- Search & filter
- Dashboard

Requirements:
- Responsive design
- Clean UI
- Error handling
- Loading states
- Production-ready
